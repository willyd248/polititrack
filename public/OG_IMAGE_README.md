# OpenGraph Image

Polititrack uses a **dynamic OpenGraph image route** at `/og` that generates images on-demand.

## Implementation

The OG image is generated by `app/og/route.tsx` using Next.js `ImageResponse` API.

**Features:**
- Dimensions: 1200x630 pixels
- Dynamic generation (no static file needed)
- Clean, minimal design with:
  - Polititrack title
  - Tagline: "Receipts-first civic clarity"
  - Subtle neutral background
  - Footer: "polititrack"

## Testing

Visit `http://localhost:3000/og` in your browser to view the generated image.

## Usage

The dynamic route is referenced in:
- `app/layout.tsx` (global default)
- `app/politician/[id]/metadata.ts` (dynamic pages)
- `app/bill/[id]/metadata.ts` (dynamic pages)
- `app/compare/layout.tsx` (compare page)
- `app/methodology/page.tsx` (methodology page)

All metadata uses `/og` as the image URL, which Next.js automatically serves from the route handler.
